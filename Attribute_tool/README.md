# Attribute Tool



## Install
```bash
conda env create -f environment.yml
```

## Run
```bash
python Attribute_GUI.py
```

## Release as exe
```bash
pyinstaller -F Attribute_GUI.py -w
```

## Hotkey for Attribute tool
a - choose pre image

d - choose next image 

z - flash attribute

c - save file
